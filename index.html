nvTrees

Roman Kogan, Texas A&M (e-mail me)

A computational system for Thompson groups F, T, V and multi-dimensional Thompson groups Vn.
  • Download: nvTrees.jar [Stand-alone Java executable]. Requires JRE v8 to run.
  • Basic instructions for using the program are here.
  • Algorithms and math background: see the poster about this project [PDF].
  • For a general introduction to Thompson groups, see CFP. The program uses a notation compatible with the examples of that paper.
  • Source code : on GitHub.
  • With extra tweaking, you can also run nvTrees as a Java applet in your browser. As Oracle is dropping support for applets, this is no longer the recommended option.
  • Screenshot:
    		Functionality:
    • Compute products of elements of Thompson groups, output reduced tree pair (group multiplication)
    • Compute conjugations, commutators, etc
    • Visualize elements of Thompson groups:
      • Tree pairs for elements of F, T, V, .., nV
      • Pattern pairs for elements of 2V
      • Rectangular diagrams for elements of F
    • Compute the growth function #(n) up to a specified value of n and given generators

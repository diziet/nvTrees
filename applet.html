nvTrees Applet
Roman Kogan, Texas A&M (e-mail me)

This applet is a calculator for the multi-dimensional Thompson groups Vn.
  • Use Firefox, Safari, or IE. The applet won't run in Chrome.
  • Download or update the Java Runtime Environment (JRE) to the latest version: Oracle Java downloads
  • Set your Java security settings to High, and add the location of this page (http://www.math.tamu.edu/~romwell/nvTrees) to the list of security exceptions, as per these instructions.
The applet will start below in this page. If the applet doesn't run, download the standalone version. Basic instructions for using the applet are here. See the poster about this project for background / explanation. Source code : on GitHub.
